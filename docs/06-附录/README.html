附录

附录部分提供高频查阅资料，适合作为阅读 C++ 工程时的手边速查表。

什么时候优先来附录

  • 你不是要系统读章节，而是正在现场查一个符号、错误或命令
  • 你已经知道自己缺什么信息，只想快速定位答案
  • 你正在排查构建、模板、宏或工具链问题

目录

  • 附录 A：核心关键字与运算符速查（Rust ↔ C++）
  • 附录 B：标准库容器与 Rust 对应表
  • 附录 C：常见编译/链接错误信息地图
  • 附录 D：C++ 版本特性速查表
  • 附录 E：阅读工具包配置指南
  • 附录 F：术语对照表

按用途快速跳转

  • 查关键字和语义近邻： 附录 A
  • 查容器映射： 附录 B
  • 查编译/链接报错： 附录 C
  • 查版本特性： 附录 D
  • 查工具命令： 附录 E
  • 查术语： 附录 F

怎么和正文配合使用

  • 正在读章节正文但卡住时，把附录当旁边的“速查抽屉”
  • 如果你是第一次系统阅读，不建议先只读附录；附录更适合作为配套而不是主线
  • 如果你已经在真实项目里，附录常常比重新通读整章更快
