## 这轮主要增强了什么

### 1. 实战章节更可执行了

- [第12章 微型 JSON 解析库](./05-第四部分-实战阅读/第12章-微型-json-解析库/README.md) 现在不只是提纲，还补了目录树、关键类型、调用链和生命周期标注。
- [第14章 驾驭大型项目](./05-第四部分-实战阅读/第14章-驾驭大型项目/README.md) 现在包含 30 分钟首轮摸底流程、阅读产物模板和风险标注方式。

### 2. 工程高风险主题更落地了

- [第6章 错误处理](./03-第二部分-边界区/第6章-错误处理/README.md) 补了 ABI / C 边界、禁用异常项目和错误翻译层。
- [第8章 模板基础](./04-第三部分-激战区/第8章-模板基础/README.md) 补了从签名反推约束、拆 `enable_if` / traits 链和模板报错回溯法。
- [第11章 未定义行为](./04-第三部分-激战区/第11章-未定义行为与隐式-unsafe/README.md) 补了高频红旗模式表和低层内存代码审查问题。

### 3. 全书模板更统一了

大部分主章节现在都有更统一的阅读结构：

- 适合谁读
- 最小示例或核心对照
- 常见误判
- 阅读时先问什么
- 练习
- 检查清单

这意味着你在不同章节之间切换时，阅读动作会更稳定。

### 4. 导航页更像真正的导航了

- [书籍首页](./README.md) 现在有“快速进入”和“按场景选路线”
- [全书总索引](./INDEX.md) 现在有“按问题快速定位”和“按投入时间阅读”
- 每个部分首页现在都补了“适合谁先读”“怎么读这一部分”“读到卡住时跳哪里”
- 每个章节 `index.md` 都补了返回部分首页和全书总索引的入口
